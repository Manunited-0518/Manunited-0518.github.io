快速排序

（基于分治的思想）

算法思想

  • 第一步：确定分界点。随机找一个元素作为分界点（一般找第一个,也可以是中间值或者随机）
  • 第二步：调整区间。让第一个区间都小于等于中间值，让第二个区间都大于等于中间值。
  • 第三步：递归处理左右两段。

第二步很关键，所以如何处理第二步是最大的问题。

这里给出一个最易于理解的方法，只是需要额外的空间，可以在忘记其他做法的时候使用此方法。

  • 第一步：建立两个数组，a[ ]和b[ ]
  • 第二步：扫描q[ l - r ]，把小于中间值的放到a，其余放到b
  • 第三步：把a放到q，再把b放到q

较为优美的方法，即为王道讲的方法。

#include <iostream>

using namespace std;

const int N = 1e6 + 10;

int n;
int q[N];

void quick_sort(int q[],int l,int r){
  if(l >=r) return;
  
  int x =q[l],i=l-1,j=r+1;
  //为什么不是i=l，j=r？
  //和之后的写法有关，此写法会将两个指针开始就向中间移动一次，所以要把指针放到数组的“两侧”
  while(i < j){
    do i++ ; while(q[i] < x);
    do j-- ; while(q[j] > x);
    if(i < j) swap(q[i],q[j]);
  }
  quick_sort(q,l,j);
  quick_sort(q,j + 1,r);
}

int main()
{
  scanf("%d",&n);
  for(int i+0; i<n ;i++) scanf("%d",&q[i]);
  
  quick_sort(q, 0, n-1);
  
  for (int i=0; i < n; i ++) printf("%d",q[i]);
  
  return 0;
}

归并排序

（基于分治的思想）

时间复杂度：nlog2n

算法思想

  • 第一步：找分界点。mid=（1+r）/2
  • 第二步：递归排序left和right
  • 第三步：归并。合二为一

最难处理的为第三步，下面是第三步的处理方法

  • 第一步：建立两个指针，分别指向两个序列的最小值
  • 第二步：比较两个指针的值，将较小者输出，并指向下一个
  • 第三步：循环执行第二步，直至一个序列的值输出完，将另一个序列的值全部输出

二分排序

整数二分

有单调性一定可以二分，但二分不一定有单调性

本质：找到一个性质，可以把区间分为两半

性质：整数二分一定有解，根据题目去判断无解的情况

算法思想

  1. 如果check是不是在左半边：

  • 求mid=(l+r+1)/2

    If( check( mid ) )

    • 若check为True: 区间为[mid,r]，更新方式为l=mid
    • 若check为false: 区间为[l,mid-1]，更新方式为r=mid-1
  1. 如果check是不是在右半边：

  • 求mid=(l+r)/2

    If( check( mid ) )

    • 若check为True: 区间为[l,mid]，更新方式为r=mid
    • 若check为false: 区间为[mid+1,r]，更新方式为l=mid+1

    其中还用到了push_back，下面是一些介绍

    push_back 是 C++ 中 std::vector 容器提供的一个成员函数，用来在 vector 的末尾添加一个元素。这个方法会将指定的元素添加到 vector 的末尾，并自动调整容器的大小（如果需要的话），以便容纳新元素。

    语法：

    1

    vector.push_back(value);

    ​ • value 是你想要添加到 vector 末尾的元素。

    特点：

    ​ • 自动扩容：当 vector 的容量不足以容纳新的元素时，push_back 会自动为 vector 分配更多的内存。

    ​ • 时间复杂度：大多数情况下，push_back 操作的时间复杂度为 O(1)，但是如果发生了扩容（当空间不足时），它的时间复杂度会是 O(n)，其中 n 是当前 vector 中元素的个数。

    C++11中添加了一种新特性——范围遍历

    C++11 引入了一种新的方式来遍历容器，称为 范围基 for 循环（range-based for loop）。它使得遍历容器变得更加简洁和易读。

    语法：

    1
2
3

    for (declaration : container) {
    // 代码块
}

    ​ • declaration 是你想要访问容器元素的声明。可以是值，也可以是引用。

    ​ • container 是你要遍历的容器，比如 vector、array、list 等。

    特点：

    ​ 1. 简洁：范围基 for 循环自动处理了迭代器或下标，避免了传统 for 循环中手动增加迭代器或下标的繁琐。

    ​ 2. 安全：对于容器元素，范围基 for 循环会确保访问的每个元素都是有效的，并且不会发生越界。

    ​ 3. 支持任何容器：不仅限于数组，vector、list、set 等容器都可以使用范围基 for 循环。

    示例：
