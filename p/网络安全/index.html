2. HTML 常见标签

2.1. meta 标签

  • <meta> 元素可提供有关页面的元信息（meta-information），比如针对搜索引擎 和更新频度的描述和关键词。

  • 设置网站关键字 <meta name=“keywords” content=“网络安全，WEB 渗透，数据安全，渗透测试， 安全培训” />

2.1.1<link>—-标签定义文档与外部资源的关系。

<link> 标签用于在 HTML 文档中引入外部资源，通常用于链接样式表（CSS），设置网站图标，或者定义其他与文档相关的信息。它是一个自闭合标签。

—-基本语法—- <link rel=“关系类型” href=“资源路径” 属性=“值”>

—-常见用法—-

  1. 引入外部样式表（CSS）

这是<link> 最常见的用途，用于将 CSS 文件引入 HTML 文档。

<link rel=“stylesheet” href=“styles.css”>

•	rel="stylesheet"：指定链接的是一个样式表。
•	href="styles.css"：指定样式表的路径。
  1. 设置网站图标（Favicon）

通过 <link> 定义网站的图标。

<link rel=“icon” href=“favicon.ico” type=“image/x-icon”>

•	rel="icon"：指定这是一个图标。
•	type="image/x-icon"：图标的文件类型。
•	href="favicon.ico"：图标文件的路径。
  1. 定义预加载资源

可以通过 <link> 提前加载关键资源以提升页面性能。

<link rel=“preload” href=“font.woff2” as=“font” type=“font/woff2” crossorigin=“anonymous”>

•	rel="preload"：提前加载资源。
•	as="font"：指定资源的类型。
•	crossorigin：如果资源需要跨域加载，需要添加此属性。
  1. 引入其他文档类型

例如，引入一个 RSS 订阅文件。

<link rel=“alternate” type=“application/rss+xml” title=“RSS” href=“feed.xml”>

  1. 引用网站的规范 URL

用于避免重复内容问题，定义此页面的标准 URL。

<link rel=“canonical” href=“https://www.example.com/page">

放置位置—-必须放在 <head> 标签内，通常作为元信息的一部分。

6.示例代码

<!DOCTYPE html> <html lang=“en”> <head> <meta charset=“UTF-8”> <meta name=“viewport” content=“width=device-width, initial-scale=1.0”> <title>示例页面</title> <link rel=“stylesheet” href=“styles.css”> <link rel=“icon” href=“favicon.ico”> </head> <body> <h1>欢迎访问！</h1> </body> </html>

注意事项

1.	自闭合：<link> 是自闭合标签，不需要 </link>。
2.	文件路径：确保 href 路径正确，支持绝对路径和相对路径。
3.	缓存问题：更新样式文件时，可在 href 中加版本号，如 styles.css?v=1.1。

<script> 标签用于在 HTML 文档中嵌入或引用 JavaScript 脚本。它可以用来实现交互、操作 DOM、动态更新内容等功能。

基本语法

<script 属性=“值”>JavaScript代码

用法

  1. 引入外部 JavaScript 文件

通过 src 属性引用外部的 JavaScript 文件。

•	src：指定 JavaScript 文件的路径。
•	文件路径可以是相对路径或绝对路径。

当脚本不依赖于页面的其他内容，可以通过 async 属性异步加载，提高页面加载速度。

•	异步加载：脚本加载完成后立即执行，不等待其他内容加载完成。
  1. 延迟加载脚本（defer 属性）

使用 defer 属性，脚本会在 HTML 解析完成后执行（但在 DOMContentLoaded 事件之前）。

•	延迟加载：保证脚本的执行顺序与页面结构一致。

通常默认是 text/javascript，但可以用于其他脚本

2.5. a 标签、img 标签、table 标签

a 标签的作用

a 标签的作用就是用于控制界面与页面之间的跳转 默认就是 self

  • self：用于在当前选项卡中跳转，也就是不新建页面跳转
  • _blank ：用于在新的选项卡中跳转，也就是新建页面跳转 Eg: <a href=“http://www.baidu.com” target=“self”>百度</a> <a href=“http://www.baidu.com” target="_blank”>百度</a> _blank 在新窗口中打开被链接文档。 _self 默认。在相同的框架中打开被链接文档。 _parent 在父框架集中打开被链接文档。 _top 在整个窗口中打开被链接文档。

2.6. 锚文本

1.基本语法

<a href="#2">锚点</a>

(1) 跳转到外部网页

(2) 跳转到页面的某个位置

a.设置目标锚点

使用 id 为目标元素标记位置：

<h2 id=“section1”>这是目标位置</h2>

b.创建锚文本链接

<a href="#section1">跳转到目标位置</a>

点击后页面会滚动到 id=“section1” 的位置

(3) 打开本地文件或资源

<a href=“files/document.pdf”>下载 PDF 文档</a>

点击后会打开或下载 document.pdf 文件，前提是文件路径正确。

(4)发送邮件

使用 mailto: 协议打开用户默认的邮件客户端：

<a href=“mailto:example@example.com”>发送邮件</a>

点击后会打开邮件客户端并准备发送邮件。

(5)拨打电话

使用 tel: 协议实现点击拨号功能（通常用于移动设备）：

<a href=“tel:+1234567890”>拨打电话</a>

点击后会启动拨号应用。

  1. 添加属性扩展功能

（1）在新窗口打开链接

通过 target="_blank" 属性设置链接在新窗口或新标签页中打开：

<a href=“https://www.example.com” target="_blank">访问 Example 网站</a>

（2）添加提示信息

使用 title 属性提供鼠标悬停时的提示：

<a href=“https://www.example.com” title=“点击访问 Example 网站”>访问 Example 网站</a>

（3）设置链接样式

结合 CSS，改变锚文本的外观：

<a href=“https://www.example.com” style=“color: red; text-decoration: none;">访问 Example 网站</a>

（4）下载文件

通过 download 属性提示浏览器下载文件而不是直接打开：

<a href=“files/document.pdf” download>下载 PDF 文档</a>

2.7. img 标签

img 元素向网页中嵌入一幅图像。 <img src=”../../../../../../../i/eg_tulip.jpg" alt=“上海鲜花港 - 郁金香” />

  • src：图片的来源（路径）。 这是必需属性，指向图片的 URL 或文件路径。
  • alt：图片的替代文本。 如果图片无法加载或被禁用，替代文本会显示在页面上，且对搜索引擎和无障碍工具（如屏幕阅读器）非常重要。 width 规定图片的高度 height 规定图片的宽度

2.10. 框架的使用

1. 基本结构

使用 <frameset> 标签代替 <body>，通过 <frame> 标签定义每个框架的内容。

cols 定义框架集中列的数目和尺寸 rows 定义框架集中行的数目和尺寸

示例：两列布局

1
2
3
4

<frameset cols="50%,50%">
  <frame src="left.html">
  <frame src="right.html">
</frameset>

说明：

• <frameset>：定义框架的布局方式，取代 <body>。 • cols：定义列布局（水平分割），可用像素、百分比或 *。 • rows：定义行布局（垂直分割），用法与 cols 类似。 • <frame>：定义每个框架加载的页面。 • src：指定每个框架加载的内容，可以是 HTML 文件或其他资源。

2. 常见布局方式

2.1 水平分割页面

1
2
3

<frame src="top.html">
<frame src="bottom.html">
</frameset>

1

  <frameset cols="50%,50%">
    <frame src="top-left.html">

  <frame src="bottom.html">
</frameset>

3. 嵌套框架

框架可以嵌套实现复杂布局。

  <frame src="header.html">

4. 添加边框、滚动和标题

框架可以通过属性设置样式：

  • frameborder：设置边框（0 表示无边框）。

  • scrolling：设置是否显示滚动条（yes、no、auto）。

  • name：给框架命名，便于其他页面的链接跳转。

示例

  <frame src="content.html" name="main" frameborder="1" scrolling="yes">
  <frame src="menu.html" name="menu" frameborder="0" scrolling="no">
</frameset>

6. 替代方案：使用现代技术

由于框架的以下问题，HTML5 废弃了 <frameset> 和 <frame>：

•	不利于 SEO（搜索引擎无法有效索引）。
•	用户体验差，不能轻松导航和书签管理。
•	兼容性问题。

推荐使用现代技术：

1.	CSS Flexbox 和 Grid Layout：布局灵活且易于控制。
2.	<iframe> 标签：加载独立页面到当前页面中。

示例：<iframe> 使用

1

<iframe src="page.html" width="100%" height="500px" frameborder="0"></iframe>
